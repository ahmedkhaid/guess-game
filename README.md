<h1>Guess the Number Challenge</h1>
<p>
  This is a simple number guessing game where the goal is to guess a hidden number between <strong>1 and 20</strong>.
</p>
<p>
  You have a maximum of <strong>20 attempts</strong> to find the correct number. The game also tracks your <strong>high score</strong> based on your best performance.
</p>
